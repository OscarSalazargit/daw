Tareas - Tema 1. Snippets VSCode

  • Se trata de investigar las opciones de los Snippets de VSCode y se propone crear el tuyo propio para HTML.
  • Existen muchos ya hechos que se pueden encontrar en GitHub y repositorios similares, no solo para HTML.
  • Se pueden crear a mano editando los ficheros JSON y con la consola del editor, pulsando F1 o Ctrl+Mayus+P y buscando Snippets:
  • También se pueden instalar extensiones que ayuden a su desarrollo y personalización como https://github.com/tahabasri/snippets
  • Por supuesto hay muchos videos y webs en Internet a modo de tutorial:
  • https://pablocianes.com/guardar-snippets-personalizados-en-visual-studio-code/
  • https://dev.to/franqsanz/como-crear-tus-propios-snippets-para-vscode-3i3g
  • La estrategia más adecuada podría ser crear distintos Snippets con la estructura completa de Header y Body con sus correspondientes Tags más habituales para distintas estructuras clásicas de web completas:
  • https://developer.mozilla.org/en-US/docs/Learn/HTML/Introduction_to_HTML/Document_and_website_structure
  • https://saimc.medium.com/html-structure-basic-d5453a2c08d5
  • https://www.programiz.com/html/layout
  • https://codewithsuraj.medium.com/html5-layout-elements-and-techniques-crafting-modern-web-pages-020bf812f706
  • https://www.webcodegeeks.com/html5/html5-page-structure-example/
  • Pero también es posible enfocarlo a trozos de código (no webs completas) realmente utilizables en muchos proyectos como crear Snippets concretos para
  • Metas en el Header para la indexación del Google bot (y 2).
  • Metas en el Header para Open Graph (y 2) para distintas redes sociales
  • Header con un logo y menú.
  • Footer con temas legales y redes sociales.

Se demostrará su funcionalidad con un enlace a un repo de GitHub donde esté el fichero y un Readme.md explicativo de su uso, puede haber un Gif animado/Vídeo mostrándolo.
